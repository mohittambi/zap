# Roadmap (example phasing)

Priorities are **not commitments** — adjust per quarter.

| Phase | Theme | Examples |
|-------|--------|----------|
| **P0** | Reliability | Fix broken links in legacy docs, migrate critical runbooks to `operations/`, ensure env template matches code |
| **P1** | Operator efficiency | Label preview, better error hints on 422 workflow actions, sync dry-run dashboards |
| **P2** | Platform | OpenAPI export, audit log MVP, performance work on API key resolution |
| **P3** | Advanced | Event-driven sync if upstream supports it, deeper mobile parity |
